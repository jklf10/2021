准备好提交了吗？

检查审阅标准

你的项目将会由优达学城的审阅专家根据 项目审阅标准 进行评估。在你提交项目之前，请使用[项目审阅标准]来检查你的项目。如果你的提交已达到所有要求，则可以提交项目。如果你发现项目还有改进的空间，请继续改进吧，直到你的项目符合审阅标准中列明的所有要求！请特别要留意，项目审阅标准中包含对代码质量的要求。在该项目中，仅仅获得正确的答案并不是最重要的，具备良好的编码习惯来得到你的解决方案也很重要。

项目提交

当你准备好，请点击“提交项目”按钮，进入项目提交页面。你可以将你的文件（html 和 ipynb 文件）作为 **a.zip **压缩文件提交。或者你可以链接到包含你项目文件的GitHub仓库。如果你使用GitHub，请注意，提交将是提交时所链接仓库的快照。建议你将每个项目保存在一个单独的仓库中，以避免任何潜在的混淆 ：如果审阅专家收到含有多个项目的文件夹，可能会对将要评估的项目有所混淆。

项目提交之后

一旦你提交了项目，你就会收到一封确认邮件。如果你对提交项目或希望检查你提交的状态有任何疑问，请及时咨询你的学习规划师。

提醒： 在上传之前，我们建议将 zip 文件以及其内部文件的文件名改为 英文 ，以尽可能避免上传过程中的问题发生。
