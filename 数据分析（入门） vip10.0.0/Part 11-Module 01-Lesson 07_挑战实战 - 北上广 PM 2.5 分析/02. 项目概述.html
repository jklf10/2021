02. 项目概述

项目概述

在本项目中，你将会使用 Python 代码来对中国五个城市（北京、上海、广州、成都、沈阳）的 PM 2.5 的值来进行分析，体验使用数据解决问题的流程。

项目指南

请查看项目中提供的 PM 2.5 的数据集，并提出几个你非常感兴趣的问题。通过调用项目中给出的帮助函数，你可以对数据进行清理、筛选、可视化、得出结论。请参考 “项目工作区” 中的具体说明来完成项目。你可以直接在工作区中的 Jupyter Notebook 来完成项目。如果你想要在本地完成项目，你可以参考 （选学）参考资料：Anaconda 和 Jupyter Notebook 来进行本地环境的安装和配置。项目文件可以从工作区或者课程资源中下载，如果有任何困难，可以咨询你的专属助教。

项目提交

在本项目中，你需要完成下面的任务：

  1. 提出问题
  2. 调用帮助函数来筛选数据
  3. 探索和可视化数据
  4. 得出结论

项目的模板已经给出，你可以直接运行已有的代码，并完成一些空格处针对你个性化探索的问题的探索。需要书写的代码之前有 ”TO DO“ 的标记。完成项目后，你需要同时提交 ipynb 和 html 文件。

审阅规范

Udacity 的项目审阅专家会根据 项目提交准则 来评估你的项目. 当你完成本项目，建议你在提交之前，对照项目评估准则进行一下自我评估。
