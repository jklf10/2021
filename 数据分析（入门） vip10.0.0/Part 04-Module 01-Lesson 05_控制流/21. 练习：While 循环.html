练习：带有 While 循环的阶乘

使用 while 循环查找数字的阶乘。

一个整数的 阶乘 是该数字乘以自身与1之间的每个整数。例如，6的阶乘（写为“ 6!”）等于6 x 5 x 4 x 3 x 2 x 1 =720。所以6! = 720。

我们可以编写一个while循环，给定任何的数字作为输入，就可以计算并输出其阶乘。

示例：如果数字（"number"）为6，则您的代码应计算并输出结果（“product”） 720。
