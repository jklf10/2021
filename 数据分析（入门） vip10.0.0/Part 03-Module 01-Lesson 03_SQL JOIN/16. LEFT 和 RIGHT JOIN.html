16. LEFT 和 RIGHT JOIN

选择以下所有表述正确的语句。
SOLUTION:
  • 如果我们更改 **FROM** 和 **JOIN** 语句中的表格，则 **LEFT JOIN** 和 **RIGHT JOIN** 的结果一样。
  • **LEFT JOIN** 将**至少**返回所有 **INNER JOIN** 返回的所有行。
  • **JOIN** 和 **INNER JOIN** 是一样的。
  • **LEFT OUTER JOIN** 和 **LEFT JOIN** 是一样的。
