22. 文本：复习 + 预告

复习

在本课程中，你学习了简单线性回归。本课程涉及的知识点有：

  1. 简单线性回归就是要建立一条回归线，为两个定量变量的关系建模。

  2. 相关系数。通过学习，你知道了相关系数是一种衡量标准，从中可了解线性关系的 强度 和 方向

  3. 可视化简单线性回归的最常用方法为散点图。

  4. 回归线由截距和斜率决定，你可以用 Python 的 statsmodels 库来找到这两个值。

  5. 学习了斜率、截距和决定系数的解释方法。
