解决方案：调试代码

代码提示用户输入10个两位数字，然后找到并打印输入的所有偶数之和。但是代码中有一个错误，因为当我输入一个数字时，我得到一个TypeError。

** 如何在代码中找到错误并进行调试？ **

  • 我添加了一个print语句来检查保存用户输入数字的变量的数据类型。print(type(userInput))。我在下一行添加了这句代码userinput = input("Enter any number: ")。
  • 输出表示:

    'class 'str' 其次是 TypeError: not all arguments converted during string formatting.
  • 这意味着变量userInput的数据类型为字符串，但我们需要将数据类型改为int。 userInput的数据类型需要更改为int。
  • 为了修复错误，我使用内置函数int()将用户输入的值转化为一个整数对象。

** 解决方案：**

user_list = []  
list_sum = 0
for i in range(10):
    userInput = int(input("Enter any number: "))
    try:
        user_list.append(userInput)
        if userInput % 2 == 0:
            list_sum += userInput
    except ValueError:
        print("Incorrect value. That's not an int!")

print("user_list: {}".format(user_list))
print("The sum of the even numbers in user_list is: {}.".format(list_sum))
