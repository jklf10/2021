31. 练习题

练习题

为了完成以下练习题，您需要在下面的工作区中用Python编写代码。这将需要你熟悉脚本编写课程中的知识点，例如读取和写入文件。你也会看到一些之前课程的知识点，希望再一次的考察和练习可以帮助你加强对它们的理解。

** 问题: **
创建一个打开flowers.txt的函数，读取其中的每一行，并将其保存为字典。主函数用于接受用户输入（用户的名字和姓氏）并解析用户输入以识别名字的第一个字母。然后用它来打印具有相同第一个字母的花名（来自第一个函数中创建的字典）。

输出示例:

>>> Enter your First [space] Last name only: Bill Newman
>>> Unique flower name with the first letter: Bellflower

** 使用工作区**

我们提供了一个带有终端和代码编辑器的编程环境，你可以在这里完成所有工作。这里有一些技巧可以帮助你快速上手。

在顶部面板上是一个代码编辑器，你可以在其中编辑Python文件** match_lower_name.py **。在此面板中向上和向下滚动以查看所有代码。你还可以通过单击并拖动其下边框来展开或缩小此面板。 我们已经上传了'flower.txt`文件，其中包含每个字母表中唯一的花名称列表。你可以在左侧单击它以在新选项卡中打开它。

在底部面板上，您可以通过单击New Terminal并在命令行上输入** python match_flower_name.py **来执行Python文件。
