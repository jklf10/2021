03. 关于诚信准则的说明

请花一点时间阅读优达学城代码诚信准则 (Udacity Honor Code)（无论你何时提交任何项目，都需要同意下列准则）：

我在此确认我提交的所有项目作业均由我个人独立完成。相应地， 如果我在项目中的部分代码片段参考了来自其他网站、书籍、论坛、博文、Github 代码库，或任何其它来源，我将正确地记录和标明出处，并解释使用它们的原因。如果未注明出处，我的项目将被优达学城官方判定为抄袭。可接受的资料来源包括：

  • 优达学城课程中的未修改或已修改代码
  • 来自具有出处的第三方来源的适量未修改或已修改代码
  • 注意：即便有出处来源，也不得引用任何其他优达学城学员项目作业中的任何内容

请不要复制其他学员作品中的任意内容。 我们会在权威系统中查出任何可疑之处，一经发现，我们会要求你重新完成并提交此项目，情节严重者可能会被移出纳米学位。 如果你在项目中途遇到困难，以下是一些在不违背“诚信准则”的前提下，寻找项目灵感的建议：

  • 首先请查看相关文档和在线教程。
  • 如果你看过其他学员的作品（在 GitHub 上或任何其他地方），请不要复制它。 请不要采用或下载其中任一部分。 你可以阅读别人的作品，了解作品背后的思维过程。
  • 思考他人作品中的设计选择 - 不同决策背后的折衷点是什么？
  • 如果需要，请花时间思考一下，因为你需要清楚如何实现你的数据分析报告。 你必须编写自己的代码，并充分理解每一步的原理。多花时间思考对于你的原创作品而言非常重要。
  • 然后返回到你的代码，并在代码中实现你自己的想法。
  • 列出你参考过的任何资料，并准确注明来源和出处。

这些建议将有助于你更好地了解如何查阅他人的作品，而不会涉嫌抄袭和剽窃。 在你的学习体验中，我们会全程相伴，帮助你取得学业上的成功。
