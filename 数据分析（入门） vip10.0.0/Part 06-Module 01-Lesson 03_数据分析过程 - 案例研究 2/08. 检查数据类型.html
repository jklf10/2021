08. 检查数据类型

# 检查数据类型
使用此 Jupyter notebook 检查每个数据集中特征的数据类型，并考虑应该做何更改，以使它们实用和（在两个数据集之间）一致。使用它来帮助你回答下面的练习问题。

Code

If you need a code on the https://github.com/udacity.

以下哪项更改，会使两个数据集中的 cyl 列更清晰和一致？
SOLUTION:
  • 从 2008 的 cyl 列中的字符串中提取整型值。
  • 将 2018 的 cyl 列中的浮点型转换为整型。

以下哪项更改会使两个数据集中的 air_pollution_score 列一致？
SOLUTION:
  • 将 2008 的列中的字符串转换为浮点型。
  • 将 2018 的列中的整型转换为浮点型。

以下哪些特征需要从字符串转换为浮点数？
SOLUTION:
  • city_mpg (2008)
  • city_mpg (2018)
  • hwy_mpg (2008)
  • hwy_mpg (2018)
  • cmb_mpg (2008)
  • cmb_mpg (2018)

哪项更改会使两个数据集中的 greenhouse_gas_score 列一致？
SOLUTION:
  • 将 2008 的列中的字符串转换为整型。
