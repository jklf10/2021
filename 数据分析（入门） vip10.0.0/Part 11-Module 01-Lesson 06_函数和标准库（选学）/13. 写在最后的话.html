13. 写在最后的话

欢迎你！你终于挑战了自我，来到 7 天试学的最后一关了，快给自己鼓鼓掌，你可能已经无形中超越了很多同龄人！

不过，只能稍微放松一下哦，如果你想抓住人才红利，高效成为一名真正的数据分析师，还需要继续前行，和我们一起快速系统性地完成整个学习路径！

正式班教什么：

在数据分析师（入门）中，我们会学到：

  • SQL 知识，了解如何从数据库中获取数据；
  • Python 的基础知识，掌握数据分析的脚本语言，也是最常用的工具；
  • 数据分析过程的流程，通过案例学习对数据的分析流程；
  • 统计学知识，学习假设检验，并学会使用 Python 来执行假设检验，并在真实场景中验证 A/B 测试的结论

在数据分析师（进阶）中，我们会学到：

  • 复习统计学中的假设检验知识，并对著名心理学现象 “斯普特鲁效应” 进行分析；
  • 通过网页、API 爬取或下载原始数据，使用 Python 进行清理；
  • 对复杂的数据集进行探索性分析（EDA），挖掘信息或者是做出预测，这里我们也会学习使用 R 语言；
  • 学习可视化的设计原则，使用 Tableau 更优雅地展示数据中的信息

透过数据，你可以：

  • 发现规律（早上 10:00 发送营销邮件打开率最高）
  • 解释现象（为什么新上线的产品功能导致次日留存 20% 的提升？）
  • 验证假设（100 万的外卖补贴应当分配 80% 在高校商圈，原因是这部分消费者对价格更加敏感）
  • 创造机会（CVS 的无糖酸奶销量连续 2 年上升，建议公司明年推出低糖款乳酸菌饮料）
  • 讲好故事（更优雅地说服老板、客户和同事）
