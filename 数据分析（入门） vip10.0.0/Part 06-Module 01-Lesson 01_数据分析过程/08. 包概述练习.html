08. 包概述练习

包练习

以下关于 Python 包的陈述哪个是正确的？
SOLUTION:
  • 你可以通过导入包来为代码添加函数和类
  • 软件包是人们编写的模块或 Python 文件的集合
  • 使用包可以简化和优化许多任务

这是使用来自包的函数的合理方式吗？

import some_package as sp
do_something()
SOLUTION: 否

这是使用来自包的函数的合理方式吗？

import some_package as sp
sp.do_something()
SOLUTION: 是

这是使用来自包的函数的合理方式吗？

import some_package as sp
some_package.do_something()
SOLUTION: 否

这是使用来自包的函数的合理方式吗？

import some_package
some_package.do_something()
SOLUTION: 是

这是使用来自包的函数的合理方式吗？

import some_package as randomlongaliasthatmakesnosense
randomlongaliasthatmakesnosense.do_something()
SOLUTION: 否
