21. 另一种字符串方法 - Split

另一种重要的字符串方法：split()

在我们处理字符串的时候，.split是非常有用的一种方法。此函数或方法返回一个名为** list **的数据容器，其中包含输入的字符串中的单词。 我们将在下一个视频中向您介绍列表（list）的概念。

split方法有两个附加的参数（* sep 和 maxsplit ）。 sep 参数表示“分隔符”。它常被用于识别字符串被分割的方式（例如空格、制表符、返回、换行符等空格字符，以及特定标点符号（例如逗号、短划线））。如果在方法中未指明 sep *参数，则默认分隔符为空格。

正如其名称一样，* maxsplit *参数表示最大分割数。该参数在新列表中给出maxsplit + 1个元素，剩余的字符串作为列表中的最后一个元素返回。你也可以在 Python 文档中了解更多如何使用该方法。

以下是.split（）方法的一些例子。

  1. 一个基本的split使用方法：
new_str = "The cow jumped over the moon."
new_str.split()```
输出:

Python
['The', 'cow', 'jumped', 'over', 'the', 'moon.']```

  1. 这里的分隔符是空格，maxsplit参数设置为3。
new_str.split(' ', 3) ```
输出:

Python
['The', 'cow', 'jumped', 'over the moon.']```

  1. 使用'.' 或句点作为分隔符。
new_str.split('.')```
输出:

Python
['The cow jumped over the moon', '']```

  1. 不使用分隔符，maxsplit参数设置为3。
new_str.split(None, 3)```
输出:

Python
['The', 'cow', 'jumped', 'over the moon.']```
