01. 简介

GitHub是目前全世界范围内最广泛使用的开源和私有软件项目托管平台，逐步成为广大开发者管理项目和发掘既有代码的首选平台。随着它普及度的不断提升，一个活跃的GitHub个人档案在求职中也起着越来越重要的参考作用，企业雇主也开始通过GitHub的代码活跃度及贡献情况来评估候选人的编程能力，尤其是针对应聘技术类职位的候选者。

在这部分课程中，你将学习如何创建及改进你的GitHub个人档案，给社群内的参与者和潜在的企业雇主留下深刻的印象。
