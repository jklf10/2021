24. 总结

祝贺您完成这节课程！

充分了解数据结构是编程和数据分析不可或缺的一部分。作为数据分析师，你将经常使用数据和代码。因此，深入了解可用的数据类型和数据结构，知道何时使用它们，将有助于你编写更高效的代码。

你可以在 HackerRank 等网站上进行更多练习。

在本课中，我们介绍了 Python 中的四个重要数据结构：

数据结构 有序性 (Ordered) 可变性 (Mutable) 构造函数 (Constructor) 举例
List Yes Yes [ ] or list() [5.7, 4, 'yes', 5.7]
Tuple Yes No ( ) or tuple() (5.7, 4, 'yes', 5.7)
Set No Yes {}* or set() {5.7, 4, 'yes'}
Dictionary No No** { } or dict() {'Jun': 75, 'Jul': 89}

*你可以使用花括号来定义这样的集合：{1,2,3}。 但是，如果你像这样将花括号留空：{}，Python 将改为创建一个空字典。因此要创建一个空集合，请使用 set（）。

**字典本身是可变的，但它的每个键都必须是不可变的。
