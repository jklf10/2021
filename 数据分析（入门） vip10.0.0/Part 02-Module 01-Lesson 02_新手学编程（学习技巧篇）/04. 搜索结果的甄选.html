04. 搜索结果的甄选

在确定关键字后，筛选搜索结果也很重要。面对成千上万条链接，哪些才是真正有用的呢？除了根据每个标题的内容及下面的摘要，进行初步判断，留意网页的来源也能够起到一定的作用。

关于编程方面的问题：

  • 概念和定义在相关的文档里都能找到答案，例如 编程语言 Python
    的官方网站 https://www.python.org/ ；Web 方面比较权威的 Mozilla Web Docs https://developer.mozilla.org/

  • 具体的应用及突发问题，活跃的论坛和开发者社区更有可能提供答案，例如 V2EX 和 Stack Overflow，中文IT社区 CSDN。

  • 而更具体教程或问题研究，可以参考一些在线教程（例如 http://www.w3school.com.cn 、 http://www.w3cschool.cn ）或者一些开发者的博客文章

当然，所有这些信息的甄选，还需要你自己去阅读和判断，而其中最重要的方法，便是亲自实践验证。
